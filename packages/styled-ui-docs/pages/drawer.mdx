# Drawer

## Import

Trend Micro Styled UI exports 5 components to help you create any Drawer.

- `Drawer`: The wrapper that provides context for its children
- `DrawerHeader`: The header that labels the Drawer
- `DrawerFooter`: The footer that houses the Drawer actions
- `DrawerBody`: The wrapper that houses the Drawer's main content
- `DrawerCloseButton`: The button that closes the Drawer.

```js
import {
  Drawer,
  DrawerHeader,
  DrawerFooter,
  DrawerBody,
  DrawerCloseButton,
} from '@trendmicro/react-styled-ui';
```

## Usage

```jsx
function BasicUsage() {
  const { isOpen, onOpen, onClose } = useDisclosure();
  return (
    <>
      <Button onClick={onOpen}>Open Drawer</Button>

      <Drawer isOpen={isOpen} onClose={onClose}>
        <DrawerHeader title="Drawer Title">Drawer Title</DrawerHeader>
        <DrawerBody>
          <Lorem count={5} />
        </DrawerBody>
        <DrawerFooter>
          <Button mr="2x" variant="primary">Submit</Button>
          <Button onClick={onClose}>
            Cancel
          </Button>
        </DrawerFooter>
      </Drawer>
    </>
  );
}
```

### Sizes

Pass the `size` prop if you need to adjust the size of the Drawer. Values can be
 `sm`, `md`, `lg`, `full` or `auto`.

```jsx
function SizeExample() {
  const { isOpen, onOpen, onClose } = useDisclosure();
  const [size, setSize] = React.useState('md');

  const handleSizeClick = newSize => {
    setSize(newSize);
    onOpen();
  };

  const sizes = ['sm', 'md', 'lg', 'full', 'auto'];

  return (
    <>
      {sizes.map(size => (
        <Button
          onClick={() => handleSizeClick(size)}
          key={size}
          m={4}
        >{`Open ${size} Drawer`}</Button>
      ))}

      <Drawer onClose={onClose} size={size} isOpen={isOpen}>
        <DrawerHeader>Drawer Title</DrawerHeader>
        <DrawerBody>
          Bla bla bla...
          Bla bla bla...
          Bla bla bla...
        </DrawerBody>
        <DrawerFooter>
          <Button onClick={onClose}>Close</Button>
        </DrawerFooter>
      </Drawer>
    </>
  );
}
```
