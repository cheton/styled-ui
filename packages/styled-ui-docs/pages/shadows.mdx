# Shadows

Add or remove shadows to elements with box-shadow utilities.

<ThemeParser theme="shadows" />

```jsx noInline
function Example() {
  const [colorMode] = useColorMode();
  const [colorStyle] = useColorStyle({ colorMode });
  const bg = colorStyle.background.primary;
  const color = colorStyle.text.primary;

  return (
    <Stack direction="row" spacing="6x" justifyContent="center">
      <Flex width={256} height={128} color={color} bg={bg} boxShadow={colorStyle.shadow.thin} align="center" justify="center">
        Thin
      </Flex>
      <Flex width={256} height={128} color={color} bg={bg} boxShadow={colorStyle.shadow.medium} align="center" justify="center">
        Medium
      </Flex>
      <Flex width={256} height={128} color={color} bg={bg} boxShadow={colorStyle.shadow.thick} align="center" justify="center">
        Thick
      </Flex>
    </Stack>
  );
}

render(<Example />);
```

## Configuration Reference

Except for breakpoints, colors, and spacing, all keys in the theme object map to one of the core themes. All keys can be replaced or extended.

For more information, see the complete properties <a href="https://styled-system.com/table" target="_blank">reference table</a>.
