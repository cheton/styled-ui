# Badge

Badges are used to highlight an item's status for quick recognition.

## Import

```js
import Badge from '@trendmicro/react-styled-ui/Badge';
// or
import { Badge } from '@trendmicro/react-styled-ui';
```

## Usage

```jsx
<Stack direction="row">
  <Badge badgeContent="5+">
    <TMIcon name="alert" />
  </Badge>
  <Badge badgeContent="99+">
    <TMIcon name="alert" />
  </Badge>
  <Badge badgeContent="87">
    <TMIcon name="alert" />
  </Badge>
  <Badge variant="dot">
    <TMIcon name="alert" />
  </Badge>
  <Badge isHidden>
    <TMIcon name="alert" />
  </Badge>
  <Badge badgeContent={0}>
    <TMIcon name="alert" />
  </Badge>
  <Badge badgeContent="!">
    <TMIcon name="alert" />
  </Badge>
  <Badge badgeContent={<TMIcon name="search-o" fontSize="xs" />}>
    <TMIcon name="alert" />
  </Badge>
</Stack>
```

### Badge Color

* Pass the `variantColor` prop to customize the color of the Badge. `variantColor`
can be any **color key with key 60** that exists in the `theme.colors`.
  > Thats mean the `green:60` exists in the `theme.colors` then can use variantColor="green".

* [Learn more about theming](./theme).

```jsx
<Stack direction="row" spacing="4x">
  <Badge badgeContent={10} />
  <Badge variantColor="orange" badgeContent={10} />
  <Badge variantColor="yellow" badgeContent={10} />
  <Badge variantColor="green" badgeContent={10} />
  <Badge variantColor="blue" badgeContent={10} />
  <Badge variantColor="gray" badgeContent={10} />
  <Badge variantColor="magenta" badgeContent={10} />
  <Badge variantColor="purple" badgeContent={10} />
  <Badge variantColor="teal" badgeContent={10} />
  <Badge variantColor="cyan" badgeContent={10} />
</Stack>
```

## Props

`Badge` composes the [`Box`](./box) component. You can override the default styles with style props.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| variant | string | 'badge' | 'badge' or 'dot'. |
| variantColor | string | 'red' | The color scheme to use for the badge. It must be a color key defined in `theme.colors`. |
| badgeContent | number\|string\|React Element  | | |
| isHidden | boolean  | false | show badge content or not. |
