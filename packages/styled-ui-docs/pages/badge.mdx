# Badge

Badges are used to highlight an item's status for quick recognition.

## Import

```js
import Badge from '@trendmicro/react-styled-ui/Badge';
// or
import { Badge } from '@trendmicro/react-styled-ui';
```

## Usage

```jsx
<Stack direction="row">
  <Badge count={10000} overflowCount={999}>
    <Skeleton variant="rect" width={42} height={42} />
  </Badge>
  <Badge dot>
    <Skeleton variant="rect" width={42} height={42} />
  </Badge>
  <Badge count={5}>
    <Skeleton variant="rect" width={42} height={42} />
  </Badge>
  <Badge count={0}>
    <Skeleton variant="rect" width={42} height={42} />
  </Badge>
  <Badge showZero count={0}>
    <Skeleton variant="rect" width={42} height={42} />
  </Badge>
</Stack>
```

### Badge Color

* Pass the `variantColor` prop to customize the color of the Badge. `variantColor`
can be any **color key with key 60** that exists in the `theme.colors`.
  > Thats mean the `green:60` exists in the `theme.colors` then can use variantColor="green".

* [Learn more about theming](./theme).

```jsx
<Stack direction="row" spacing="4x">
  <Badge count={10} />
  <Badge variantColor="green" count={10} />
  <Badge variantColor="red" count={10} />
  <Badge variantColor="blue" count={10} />
  <Badge variantColor="magenta" count={10} />
  <Badge variantColor="purple" count={10} />
  <Badge variantColor="teal" count={10} />
  <Badge variantColor="cyan" count={10} />
</Stack>
```

* Example

```jsx
<Flex alignItems="center">
  <Text size="xl" fontWeight="bold">
    Trend Micro Frontend
  </Text>
  <Badge ml="1x" variantColor="green">
    New
  </Badge>
</Flex>
```

## Props

`Badge` composes the [`Box`](./box) component. You can override the default styles with style props.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| variantColor | string | 'gray' | The color scheme to use for the badge. It must be a color key defined in `theme.colors`. |
