# Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

## Import

```js
import Alert from '@trendmicro/react-styled-ui/Alert';
// or
import { Alert } from '@trendmicro/react-styled-ui';
```

## Usage

### Severity levels

The alert offers four severity levels, each has a distinctive icon and color.

```jsx
<Stack direction="column" spacing="4x">
  <Alert severity="success">
    This is a success alert.
  </Alert>
  <Alert severity="info">
    This is an info alert.
  </Alert>
  <Alert severity="warning">
    This is a warning alert.
  </Alert>
  <Alert severity="error">
    This is an error alert.
  </Alert>
</Stack>
```

### Multiline text

```jsx noInline
const Paragraph = (props) => (
  <Text display="block" mb="1x" {...props} />
);

function Example() {
  return (
    <Alert severity="info">
      <Paragraph>This is the first paragraph.</Paragraph>
      <Paragraph mb={0}>This is the second paragraph.</Paragraph>
    </Alert>
  );
}

render(<Example />);
```

### Formatted title

You can use the `Heading` component to display a formatted title above the content.

```jsx
<Stack direction="column" spacing="4x">
  <Alert severity="success">
    <Heading fontWeight="semibold">Success</Heading>
    This is a success alert.
  </Alert>
  <Alert severity="info">
    <Heading fontWeight="semibold">Info</Heading>
    This is an info alert.
  </Alert>
  <Alert severity="warning">
    <Heading fontWeight="semibold">Warning</Heading>
    This is a warning alert.
  </Alert>
  <Alert severity="error">
    <Heading fontWeight="semibold">Error</Heading>
    This is an error alert.
  </Alert>
</Stack>
```

### Actions

An alert can have an action, such as a close or action button. It is rendered after the message, at the end of the alert.

```jsx
<Stack direction="column" spacing={1}>
  <Alert severity="warning">
    <Flex justify="space-between">
      <Text>Your browser is currently not supported. <Link>More Information</Link></Text>
      <ButtonBase opacity=".6" _hover={{ opacity: 1 }}>
        <TMIcon name="close-s" />
      </ButtonBase>
    </Flex>
  </Alert>
  <Alert severity="error">
    <Flex justify="space-between">
      <Text>Your license will expire in 7 days.</Text>
      <Flex>
        <FlatButton size="sm" variant="outline" variantColor="black:primary">
          Renew Now
        </FlatButton>
        <Space width="4x" />
        <ButtonBase opacity=".6" _hover={{ opacity: 1 }}>
          <TMIcon name="close-s" />
        </ButtonBase>
      </Flex>
    </Flex>
  </Alert>
</Stack>
```

### Transition

You can use a transition component such as `Collapse` to give the appearance of a smooth transition.

```jsx
function Example() {
  const [isOpen, setIsOpen] = React.useState(true);
  const handleOpen = () => setIsOpen(true);
  const handleClose = () => setIsOpen(false);

  return (
    <>
      <Collapse isOpen={isOpen}>
        <Alert severity="info">
          <Flex justify="space-between">
            Click the close button on the right side.
            <ButtonBase onClick={handleClose} opacity=".6" _hover={{ opacity: 1 }}>
              <TMIcon name="close-s" />
            </ButtonBase>
          </Flex>
        </Alert>
      </Collapse>
      <Box mt="4x">
        <Button onClick={handleOpen} disabled={isOpen}>
          Reopen
        </Button>
      </Box>
    </>
  );
}
```

### Icons

The `icon` prop allows you to override the default icon for the specified severity.

Setting the `icon` prop to `false` will remove the icon altogether.

```jsx
<Stack direction="column" spacing="4x">
  <Alert severity="success">
    This is a success alert.
  </Alert>
  <Alert severity="success" icon="_core.severity-success">
    This is a success alert.
  </Alert>
  <Alert
    severity="success"
    icon={<TMIcon name="check-circle-o" lineHeight={1} color="gray:80" />}
  >
    This is a success alert.
  </Alert>
  <Alert severity="success" icon={false}>
    This is a success alert.
  </Alert>
</Stack>
```

## Props

`Alert` composes the `Flex` component.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| severity | string | 'success' | The severity level of the alert. One of: `'success'`, `'info'`, `'warning'`, `'error'` |
| icon | string \| ReactNode \| false | | Override the icon displayed before the children. Unless provided, the icon is mapped to the value of the `severity` prop. |
