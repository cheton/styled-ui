# ButtonGroup

Group a series of [buttons](./button) together on a single line with the button group.

## Import

```js
import ButtonGroup from '@trendmicro/react-styled-core/ButtonGroup';
// or
import { ButtonGroup } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<ButtonGroup>
  <Button variant="solid">Button</Button>
  <Button variant="outline">Button</Button>
</ButtonGroup>
```

### Buttons Variant

Use the `variant` prop to change the visual style of the Button. You can set the value to `solid`, `outline`. Use `outline` to remove all background images and colors on any button.

```jsx
<ButtonGroup variant="outline">
  <Button>Button</Button>
  <Button>Button</Button>
</ButtonGroup>
```

### Buttons Color

Use the `variantColor` prop to change the color scheme of the Button. `variantColor` can be any color key with key `50`(hover), `60`(normal) and `70`(active) that exist in the them.color.

```jsx
<ButtonGroup variantColor="red">
  <Button>Emphasis Button</Button>
  <Button>Primary Button</Button>
  <Button variantColor="gray">Default Button</Button>
</ButtonGroup>
```

### Buttons Sizes

Use the `size` prop to change the size of the button. You can set the value to `sm`, `md`, or `lg`.

```jsx
<ButtonGroup size="lg">
  <Button size="sm">Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</ButtonGroup>
```

### isAttached

Use the `isAttached` prop to change the borderRadius of button that are direct children will be altered to look flushed together.

```jsx
<ButtonGroup variant="outline" isAttached="true">
  <Button>Left</Button>
  <Button>Middle</Button>
  <Button>Right</Button>
</ButtonGroup>
```

### spacing

Use the `spacing` prop to change the right margin between Buttons in the group. `spacing` can be any key that exist in the them.space.

```jsx
<ButtonGroup spacing="lg">
  <Button>Left</Button>
  <Button>Middle</Button>
  <Button>Right</Button>
</ButtonGroup>
```

## Props

The ButtonGroup composes the `Box` component so you can pass props for `Box`. These are props related to the ButtonGroup component.

| Name               | Type                 | Default       | Description                                                            |
| ------------------ | -------------------- | ------------- | ---------------------------------------------------------------------- |
| `isAttached`       | boolean              | false         | If `true`, the borderRadius of button that are direct children will be altered to look flushed together.  |
| `size`             | `sm`, `md`, `lg`     | `md`          | Sets the size for all Buttons in the group.                            |
| `spacing`          | string               | `xs`          | The right margin between Buttons in the group. Use a key passed in `theme.space`.  |
| `variant`          | `solid`, `outline`   | `solid`       | Sets the variant of all Buttons in the group.                          |
| `variantColor`     | string               |               | Sets the variant of all Buttons in the group. Use a color key passed in `theme.colors`.  |
