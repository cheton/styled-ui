# ButtonGroup

ButtonGroup is used to group related buttons.

## Import

```js
import ButtonGroup from '@trendmicro/react-styled-core/ButtonGroup';
// or
import { ButtonGroup } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<ButtonGroup>
  <Button variant="solid">One</Button>
  <Button variant="outline">Two</Button>
</ButtonGroup>
```

### Buttons Variant

Use the `variant` prop to change the visual style of every button in a group. You can set the value to `solid`, `outline`. Use `outline` to remove all background images and colors on any button.

```jsx
<>
  <ButtonGroup variant="solid">
    <Button>One</Button>
    <Button>Two</Button>
  </ButtonGroup>
  <Space width="1rem" />
  <ButtonGroup variant="outline">
    <Button>One</Button>
    <Button>Two</Button>
  </ButtonGroup>
</>
```

### Buttons Color

Use the `variantColor` prop to change the color scheme of every button in a group. `variantColor` can be any color key with key `50`(hover), `60`(normal) and `70`(active) that exist in the `theme.color`.

```jsx
<ButtonGroup variantColor="blue">
  <Button>Left</Button>
  <Button>Middle</Button>
  <Button>Right</Button>
</ButtonGroup>
```

### Buttons Size

Use the `size` prop to change the size of every button in a group. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack spacing="1rem">
  <ButtonGroup variant="outline" size="sm">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
  <ButtonGroup variant="outline" size="md">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
  <ButtonGroup variant="outline" size="lg">
    <Button>Left</Button>
    <Button>Middle</Button>
    <Button>Right</Button>
  </ButtonGroup>
</Stack>


```

## Props

The ButtonGroup composes the `Box` component so you can pass props for `Box`. These are props related to the ButtonGroup component.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| `size` | `sm`, `md`, `lg` | `md` | Sets the size for all Buttons in the group. |
| `variant` | `solid`, `outline` | `solid` | Sets the variant of all Buttons in the group. |
| `variantColor` | string | | Sets the variant of all Buttons in the group. Use a color key passed in `theme.colors`. |
