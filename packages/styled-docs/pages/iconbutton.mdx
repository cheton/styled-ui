# Icon Button

`IconButton` composes the [`ButtonBase`](./buttonbase) component, except that it renders only an icon.

## Import

```js
import IconButton from '@trendmicro/react-styled-core/IconButton';
// or
import { IconButton } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<IconButton icon="_core.warning-triangle" />
```

### Colors

```jsx
<Stack direction="row" spacing="4x">
  <IconButton icon="_core.warning-triangle" color="red" />
  <IconButton icon="_core.warning-triangle" color="yellow" />
  <IconButton icon="_core.warning-triangle" color="green" />
</Stack>
```

### Sizing

```jsx
<IconButton
  icon="_core.check"
  iconSize="10x"
  px="4x"
  py="4x"
  border={1}
  borderColor="gray:50"
  borderRadius="circle"
/>
```

## Props

| Name | Type | Default | Description |
| :--- | :--- | ------- | ----------- |
| icon | string | | The icon to be used. Refer to the [Icons](./icon) section of the docs for the available icon options. |
| iconSize | string | | The size of the icon. |
