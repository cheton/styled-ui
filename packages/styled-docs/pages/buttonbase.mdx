# ButtonBase

The ButtonBase component does not contains any styles. It aims to be a simple building block for creating a button.

## Import

```js
import ButtonBase from '@trendmicro/react-styled-core/ButtonBase';
// or
import { ButtonBase } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<ButtonBase>Base Button</ButtonBase>
```

## Props

The ButtonBase component composes [PseudoBox](./pseudobox) and all props you pass are converted to style props. This means you can override any style of the ButtonBase via props.
