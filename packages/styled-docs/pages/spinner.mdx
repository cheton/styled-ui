# Spinner

A spinner can be used to display the loading state when the part of the page is waiting for the results.

## Import

```js
import Spinner from '@trendmicro/react-styled-core/Spinner';
// or
import { Spinner } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Spinner />
```

### Colors

```jsx
<Stack direction="row" spacing="6x">
  <Spinner color="red:50" />
  <Spinner color="red" />
  <Spinner color="white" />
  <Spinner color="rgba(255, 167, 196, 1)" />
  <Spinner color="#8f41e9" />
</Stack>
```

### Sizes

```jsx
<Stack direction="row" spacing="6x">
  <Spinner size="xs" />
  <Spinner size="sm" />
  <Spinner size="md" />
  <Spinner size="lg" />
  <Spinner size="xl" />
</Stack>
```

### StrokeWidth

```jsx
<Stack direction="row" spacing="6x">
  <Spinner strokeWidth={1} />
  <Spinner strokeWidth={2} />
  <Spinner strokeWidth={3} />
  <Spinner strokeWidth={5} />
  <Spinner strokeWidth={10} />
</Stack>
```


## Props

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| `size` | `string` | 'md' | The size of the spinner. One of: 'xl', 'lg', 'md', 'sm', 'xs' |
| `color` | `string` | 'blue:60' | The color of the spinner. |
| `strokeWidth` | `number` | depand on size | One of: 'xl': 4, 'lg': 4, 'md': 4, 'sm': 2, 'xs': 2 |
