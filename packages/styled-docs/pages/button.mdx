# Button

`Button` is used to trigger an action or event, such as submitting a form, opening a [`Modal`](./modal), canceling an action, or performing a delete operation.

## Import

```js
import Button from '@trendmicro/react-styled-core/Button';
// or
import { Button } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Button>Default Button</Button>
```

### Variants

Use the `variant` prop to change the visual style of the Button. You can set the value to `solid`, `outline`, `emphasis`, `primary`, `default`, `secondary`, `ghost`.

```jsx
<Stack direction="row" spacing="2x" shouldWrapChildren>
  <Button variant="solid">Solid Button</Button>
  <Button variant="outline">Outlined Button</Button>
  <Button variant="emphasis">Emphasis Button</Button>
  <Button variant="primary">Primary Button</Button>
  <Button variant="default">Default Button</Button>
  <Button variant="secondary">Secondary Button</Button>
  <Button variant="ghost">Ghost Button</Button>
</Stack>
```

### Colors

Use the `variantColor` prop to change the color scheme of the Button. `variantColor` can be any color key exist in the them.color.

The color scheme for `emphasis`, `primary`, `default`, `secondary`, `ghost` buttons cannot be changed via `variantColor`.

| variant | normal | hover | active | focus | disabled | font color |
| :------ | :----- | :---- | :----- | :---- | :------- | :--------- |
| solid | `${variantColor}:60` | `${variantColor}:50` | `${variantColor}:70` | `blud:40` | `gray:60` | `white` |
| outline | `${variantColor}:60` | `${variantColor}:50` | `${variantColor}:50` | `blue:60` | `gray:60` | `${variantColor}:40` |
| emphasis | `red:60` | `red:50` | `red:70` | `blue:60` | `gray:60` | `white` |
| primary | `blue:60` | `blue:50` | `blue:70` | `blue:60` | `gray:60` | `white` |
| default | `gray:60` | `gray:50` | `gray:70` | `blue:60` | `gray:60` | `white` |
| secondary | `gray:60` | `blue:50` | `blue:70` | `blue:60` | `gray:60` | `white`/`black`, `blue:40` |
| ghost | | `blue:50` | `blue:70` | `blue:60` | `gray:60` | `white`/`black`, `blue:40` |

```jsx
<Stack direction="column" spacing="4x">
  <Stack direction="row" spacing="2x" alignItems="center" shouldWrapChildren>
    <Button variantColor="red">Button</Button>
    <Button variantColor="blue">Button</Button>
    <Button variantColor="gray">Button</Button>
  </Stack>
  <Stack direction="row" spacing="2x" alignItems="center" shouldWrapChildren>
    <Button variant="outline" variantColor="red">Button</Button>
    <Button variant="outline" variantColor="blue">Button</Button>
    <Button variant="outline" variantColor="gray">Button</Button>
  </Stack>
</Stack>
```

### Sizes

Use the `size` prop to change the size of the button. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack direction="column" spacing="4x">
  <Stack direction="row" spacing="2x" alignItems="center" shouldWrapChildren>
    <Button size="sm">Small</Button>
    <Button size="md">Medium</Button>
    <Button size="lg">Large</Button>
  </Stack>
  <Stack direction="row" spacing="2x" alignItems="center" shouldWrapChildren>
    <Button variant="outline" size="sm" variantColor="gray">Small</Button>
    <Button variant="outline" size="md" variantColor="gray">Medium</Button>
    <Button variant="outline" size="lg" variantColor="gray">Large</Button>
  </Stack>
  <Stack direction="row" spacing="2x" alignItems="center" shouldWrapChildren>
    <Button variant="default" size="sm">Small</Button>
    <Button variant="default" size="md">Medium</Button>
    <Button variant="default" size="lg">Large</Button>
  </Stack>
</Stack>
```

```jsx
  </Stack>

  </Stack>

```

### Icon buttons

Icon buttons are commonly used in toolbars. Sometimes you might want to add label to certain icon buttons to enhance the UX. You can use [Space](./space) to add a padding space with `.5rem` (`8px`) between icon and label.

```jsx

  <Stack direction="row" spacing="2x">
    <Button variantColor="blue" borderRadius="circle" width="8x" height="8x">
      <TMIcon name="add" color="white" />
    </Button>
    <Button variantColor="blue" width="8x" height="8x">
      <TMIcon name="add" color="white" />
    </Button>
    <Button variantColor="blue" borderRadius="2rem">
      <TMIcon name="add" color="white"/>
      <Space width=".5rem" />
      Add Account
    </Button>
    <Button variantColor="blue">
      <TMIcon name="add" color="white"/>
      <Space width=".5rem" />
      Add Account
    </Button>
  </Stack>
  <Stack direction="row" spacing="2x">
    <Button variant="outline" borderRadius="circle" width="8x" height="8x">
      <TMIcon name="add" />
    </Button>
    <Button variant="outline" width="8x" height="8x">
      <TMIcon name="add" />
    </Button>
    <Button variant="outline" borderRadius="2rem">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
    <Button variant="outline">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
  </Stack>
  <Stack direction="row" spacing="2x">
    <Button variant="primary" width="8x" height="8x" px="0"  borderRadius="circle">
      <TMIcon name="add" />
    </Button>
    <Button variant="primary" width="8x" height="8x" px="0">
      <TMIcon name="add" />
    </Button>
    <Button variant="primary" borderRadius="2rem">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
    <Button variant="primary">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
  </Stack>
</Stack>
```

## Custom button

In case you need to customize the button, pass style props to the [ButtonBase](./buttonbase) component to create a custom button.

```jsx
<ButtonBase
  bg="#f5f6f7"
  border="1"
  borderColor="#ccd0d5"
  borderRadius="md"
  color="#4b4f56"
  fontSize="sm"
  fontWeight="semibold"
  height="24px"
  lineHeight="sm"
  px="2x"
  transition="all 0.2s cubic-bezier(.08,.52,.52,1)"
  _hover={{
    bg: "#ebedf0"
  }}
  _active={{
    bg: "#dddfe2",
    transform: "scale(0.98)",
    borderColor: "#bec3c9",
  }}
  _focus={{
    boxShadow:
      "0 0 1px 2px rgba(88, 144, 255, .75), 0 1px 1px rgba(0, 0, 0, .15)",
  }}
>
  Join Group
</ButtonBase>
```

## Props

`Button` composes the [`ButtonBase`](./buttonbase) component. You can override the default styles with style props.

| Name | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| disabled | `boolean` | false | If `true`, the button will be disabled. |
| size | `string` | `md` | The size of the button. One of: `sm`, `md`, `lg` |
| variant | `string` | `solid` | The variant of the button style to use. One of: `solid`, `outline`, `emphasis`, `primary`, `default`, `secondary`, `ghost` |
| variantColor | `string` | | The color of the button. It must be a color key defined in `theme.colors`. |
