# Button

Button is used to trigger an action or event, such as submitting a form, opening a [Dialog](./modal), canceling an action, or performing a delete operation.

## Import

```js
import Button from '@trendmicro/react-styled-core/Button';
// or
import { Button } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Button>Default Button</Button>
```

### Variants

Use the `variant` prop to change the visual style of the Button. You can set the value to `solid`, `outline`. Use `outline` to remove all background images and colors on any button.

```jsx
<Stack direction="row" spacing="1rem" alignItems="center" shouldWrapChildren>
  <Button variant="solid">Solid Button</Button>
  <Button variant="outline">Outlined Button</Button>
</Stack>
```

### Colors

Use the `variantColor` prop to change the color scheme of the Button. `variantColor` can be any color key with key `50`(hover), `60`(normal) and `70`(active) that exist in the them.color.

```jsx
<Stack direction="row" spacing="1rem" alignItems="center" shouldWrapChildren>
  <Button variantColor="red">Emphasis Button</Button>
  <Button variantColor="blue">Primary Button</Button>
  <Button variantColor="gray">Default Button</Button>
</Stack>
```

### Sizes

Use the `size` prop to change the size of the button. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack direction="row" spacing="1rem" alignItems="center" shouldWrapChildren>
  <Button size="sm">Small</Button>
  <Button size="md">Medium</Button>
  <Button size="lg">Large</Button>
</Stack>
```

### `REVIEW REQUIRED` Text buttons

Text buttons are typically used for less-pronounced actions.

```jsx noInline
const TextButton = (props) => (
  <Button variant="none" border="none" {...props} />
);

function Example() {
  return (
    <>
      <TextButton>Default</TextButton>
      <TextButton color="red.50">Emphasis</TextButton>
      <TextButton color="blue.50">Primary</TextButton>
      <TextButton disabled>Disabled</TextButton>
    </>
  );
}

render(<Example />);
```

### `REVIEW REQUIRED` Icon buttons

Icon buttons are commonly used in toolbars. 

```jsx noInline
const PlainButton = (props) => (
  <Button
    variant="outline"
    border="none"
    {...props}
  />
);

function Example() {
  return (
    <Flex>
      <PlainButton>
        <TMIcon name="add" />
      </PlainButton>
      <PlainButton>
        <TMIcon name="minus" />
      </PlainButton>
    </Flex>
  );
}

render(<Example />);
```

Sometimes you might want to add label to certain icon buttons to enhance the UX. You can use [Space](./space) to add a padding space with `.5rem` (`8px`) between icon and label.

```jsx
<Stack direction="column" spacing="1rem">
  <Stack direction="row" spacing="1rem">
    <Button variantColor="blue" borderRadius="circle" width="2rem" height="2rem">
      <TMIcon name="add" color="white" />
    </Button>
    <Button variantColor="blue" width="2rem" height="2rem">
      <TMIcon name="add" color="white" />
    </Button>
    <Button variantColor="blue" borderRadius="2rem">
      <TMIcon name="add" color="white"/>
      <Space width=".5rem" />
      Add Account
    </Button>
    <Button variantColor="blue">
      <TMIcon name="add" color="white"/>
      <Space width=".5rem" />
      Add Account
    </Button>
  </Stack>
  <Stack direction="row" spacing="1rem">
    <Button variant="outline" borderRadius="circle" width="2rem" height="2rem">
      <TMIcon name="add" />
    </Button>
    <Button variant="outline" width="2rem" height="2rem">
      <TMIcon name="add" />
    </Button>
    <Button variant="outline" borderRadius="2rem">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
    <Button variant="outline">
      <TMIcon name="add" />
      <Space width=".5rem" />
      Add Account
    </Button>
  </Stack>
</Stack>
```

## Custom button

In event you need to make your own custom button, you can leverage the [ButtonBase](./buttonbase) component.

```jsx
<ButtonBase
  bg="#f5f6f7"
  border="1px solid"
  borderColor="#ccd0d5"
  borderRadius="md"
  color="#4b4f56"
  fontSize="14px"
  fontWeight="semibold"
  height="24px"
  lineHeight="1.2"
  px="8px"
  transition="all 0.2s cubic-bezier(.08,.52,.52,1)"
  _hover={{
    bg: "#ebedf0"
  }}
  _active={{
    bg: "#dddfe2",
    transform: "scale(0.98)",
    borderColor: "#bec3c9",
  }}
  _focus={{
    boxShadow:
      "0 0 1px 2px rgba(88, 144, 255, .75), 0 1px 1px rgba(0, 0, 0, .15)",
  }}
>
  Join Group
</ButtonBase>
```

## Props

Button composes [ButtonBase](./buttonbase). You can override the default styles with style props.

| Name               | Type                 | Default       | Description                                                            |
| ------------------ | -------------------- | ------------- | ---------------------------------------------------------------------- |
| `disabled`         | boolean              | `false`       | If `true`, the button will be disabled.                                |
| `size`             | `sm`, `md`, `lg`     | `md`          | The size of the button.                                                |
| `variant`          | `solid`, `outline`   | `solid`       | The variant of the button style to use.                                |
| `variantColor`     | string               |               | The color of the button. Use a color key passed in `theme.colors`.     |
