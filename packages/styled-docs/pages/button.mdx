# Button

The Button component is used to trigger an action or event, such as submitting a form, opening a [Dialog](./modal), canceling an action, or performing a delete operation.

## Import

```js
import Button from '@trendmicro/react-styled-core/Button';
// or
import { Button } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Button>Default Button</Button>
```

### Button Variant

Use the `variant` prop to change the visual style of the Button. You can set the value to `solid`, `outline`. Use `outline` to remove all background images and colors on any button.

```jsx
<Stack direction="row" spacing="0.5rem">
  <Button variant="solid">Button</Button>
  <Button variant="outline">Button</Button>
</Stack>
```

### Button Color

Use the `variantColor` prop to change the color scheme of the Button. `variantColor` can be any color key with key `50`(hover), `60`(normal) and `70`(active) that exist in the them.color.

```jsx
<Stack direction="row" spacing="0.5rem">
  <Button variantColor="red">Emphasis Button</Button>
  <Button variantColor="blue">Primary Button</Button>
  <Button variantColor="gray">Default Button</Button>
</Stack>
```

### Button Sizes

Use the `size` prop to change the size of the button. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack direction="row" spacing="0.5rem" shouldWrapChildren={true} alignItems="center">
  <Button size="sm">Button</Button>
  <Button size="md">Button</Button>
  <Button size="lg">Button</Button>
</Stack>
```

## Accessibility

- Button has `role` button.
- When Button has <kbd>focus</kbd>, <kbd>Space</kbd> and <kbd>Enter</kbd> activates it.

## Composition

The Button component composes [ButtonBase](./buttonbase), and you can override any style of the Button via props.

```jsx
<Button
  variant="outline"
  height="48px"
  width="200px"
  border="none"
>
  Ghost Button
</Button>
```
### Button with Icon

You can add icons to the Button components.

```jsx
<Stack spacing="0.5rem">
  <Stack direction="row" spacing="0.5rem">
    <Button variantColor="blue" borderRadius="circle" px="0px">
      <TMIcon name="add" color="white" width="32px" />
    </Button>
    <Button variantColor="blue" px="0px">
      <TMIcon name="add" color="white" width="32px"/>
    </Button>
    <Button variantColor="blue" borderRadius="32px"><TMIcon name="add" color="white"/>Add Account</Button>
    <Button variantColor="blue"><TMIcon name="add" color="white"/>Add Account</Button>
  </Stack>
  <Stack direction="row" spacing="0.5rem">
    <Button variant="outline" borderRadius="circle" px="0px">
      <TMIcon name="add" width="32px"/>
    </Button>
    <Button variant="outline" px="0px">
      <TMIcon name="add" width="32px" />
    </Button>
    <Button variant="outline" borderRadius="32px"><TMIcon name="add"/>Add Account</Button>
    <Button variant="outline"><TMIcon name="add"/>Add Account</Button>
  </Stack>
</Stack>
```

## Custom Button

In event you need to make your own custom button, you can leverage the [ButtonBase](./buttonbase) component.

```jsx
<ButtonBase
  bg="#f5f6f7"
  border="1px solid"
  borderColor="#ccd0d5"
  borderRadius="md"
  color="#4b4f56"
  fontSize="14px"
  fontWeight="semibold"
  height="24px"
  lineHeight="1.2"
  px="8px"
  transition="all 0.2s cubic-bezier(.08,.52,.52,1)"

  _hover={{
    bg: "#ebedf0"
  }}
  _active={{
    bg: "#dddfe2",
    transform: "scale(0.98)",
    borderColor: "#bec3c9",
  }}
  _focus={{
    boxShadow:
      "0 0 1px 2px rgba(88, 144, 255, .75), 0 1px 1px rgba(0, 0, 0, .15)",
  }}
>
  Join Group
</ButtonBase>
```

## Props

The Button composes the `PseudoBox` component so you can pass props for `PseudoBox`. These are props related to the Button component.

| Name               | Type                 | Default       | Description                                                            |
| ------------------ | -------------------- | ------------- | ---------------------------------------------------------------------- |
| `disabled`         | boolean              | `false`       | If `true`, the button will be disabled.                                |
| `size`             | `sm`, `md`, `lg`     | `md`          | The size of the button.                                                |
| `variant`          | `solid`, `outline`   | `solid`       | The variant of the button style to use.                                |
| `variantColor`     | string               |               | The color of the button. Use a color key passed in `theme.colors`.     |
