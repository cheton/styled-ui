# Input

```js
import Input from '@trendmicro/react-styled-core/Input';
// or
import { Input } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Input
  placeholder="Placeholder text"
/>
```

### Sizes

Use the `size` prop to change the size of the `Input`. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack direction="column" spacing="4x">
  <Input size="sm" placeholder="Placeholder text" />
  <Input size="md" placeholder="Placeholder text" />
  <Input size="lg" placeholder="Placeholder text" />
</Stack>
```
