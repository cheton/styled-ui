# Input

```js
import Input from '@trendmicro/react-styled-core/Input';
// or
import { Input } from '@trendmicro/react-styled-core';
```

## Usage

```jsx
<Input placeholder="Basic usage" />
```

### Disabled input

```jsx
<Stack direction="column" spacing="4x">
  <Input disabled placeholder="Placeholder text" />
  <Input disabled defaultValue="Input text" placeholder="Placeholder text" />
</Stack>
```

### Sizes

Use the `size` prop to change the size of the `Input`. You can set the value to `sm`, `md`, or `lg`.

```jsx
<Stack direction="column" spacing="4x">
  <Input size="sm" placeholder="Small size (24px)" />
  <Input size="md" placeholder="Default size (32px)" />
  <Input size="lg" placeholder="Large size (40px)" />
</Stack>
```

### Variants

```jsx
<Stack direction="column" spacing="4x">
  <Input variant="outline" placeholder="Outline" />
  <Input variant="unstyled" placeholder="Unstyled" />
</Stack>
```

### Validation

The `error` prop toggles the error state:

```jsx
function Example() {
  const InlineError = (props) => (
    <Text fontSize="sm" lineHeight="sm" color="red:50" {...props} />
  );

  return (
    <>
      <Input error placeholder="Enter your name here" mb="1x" />
      <InlineError>
        This is a required field.
      </InlineError>
    </>
  );
}
```
